Welcome to My Storage Space

This page was created to talk about my 3rd Year Project of my undergraduate BSc in Music Technology. The project is based around implementing granular synthesis on an Arduino ATmega328 board, and making some form of tangible (and hopefully musical) instrument from it. GitHub will be hosting the code I write as firmware for the instrument, and probably some schematics and board layouts towards the end of the project.
